# 中国 A 股量化交易与金融百科教程

从 [00 中国 A 股量化交易完整学习路线](./00-完整教程索引与学习路线.md) 开始。

教程已合并为 19 篇扁平化笔记：`00` 为总索引，`01～18` 从金融基础、A 股数据、策略回测一路讲到本地 Web 研究平台。文件名同时包含顺序和主题，不再使用“数字文件夹 + 数字文件”的冗余结构。

## 文件位置

- Obsidian 笔记：`docs`。
- Quant Lab 项目：`quant-lab`。
- 项目启动说明：[`quant-lab/README.md`](../quant-lab/README.md)。

后端全部使用 Python 3.12、FastAPI、Pandas、SQLAlchemy 和 pytest，不使用 Java、Spring、Maven 或 Gradle。Vue 3 与 ECharts 只负责浏览器前端。

> 所有股票、行情、费率和结果均用于教学。项目不连接券商、不发送真实订单，不构成投资建议或收益承诺。
