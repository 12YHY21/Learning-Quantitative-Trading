## 为什么先区分资产

行情软件里每个资产看起来都有代码和价格，但法律权利、现金流、到期方式和损失边界不同。若把期货当股票处理，会漏掉保证金和到期；若把 ETF 当单只公司，会误解跟踪误差和申赎机制。

## 六类常见金融资产

### 股票

股票代表对公司的所有者权益。股东获得剩余收益：公司先支付员工、供应商、税费和债权人，剩余价值才属于股东。因此上行空间可能很大，损失也可能接近全部本金。

### 债券

债券是发行人向投资者借款形成的债权凭证，通常约定利息和到期偿还。债券仍有信用风险、利率风险、流动性风险和再投资风险。

### 基金与 ETF

基金把许多投资者的资金集中投资。ETF 是 Exchange-Traded Fund，中文为交易型开放式指数基金，可像股票一样交易。成交价格可能偏离基金净值，指数上涨也不保证 ETF 完全同步，因为还有费用、复制方法和跟踪误差。

### 期货

期货是标准化远期合约，通常使用保证金。保证金不是资产价格，而是履约担保。用较少保证金控制较大名义金额会形成杠杆，盈利和亏损都被放大。

### 期权

期权买方支付权利金，获得在约定条件下买入或卖出标的的权利。期权价值受标的价格、行权价、期限、波动率和利率共同影响。

### 现金与存款

现金名义价值稳定，但受通胀影响实际购买力。现金也是组合中的主动配置，因为它提供流动性。

## 收益从哪里来

股票总收益不能只看价格：

$$
R_{\text{total}}=
\frac{P_1-P_0+D}{P_0}
$$

$P_0$ 是期初价格，$P_1$ 是期末价格，$D$ 是期间每股分红。10 元买入、期末 10.5 元、分红 0.2 元：

$$
R_{\text{total}}=\frac{10.5-10+0.2}{10}=7\%
$$

只算价格会得到 5%，漏掉分红。

| 资产 | 主要收益来源 | 主要风险 |
|---|---|---|
| 股票 | 股价变化、分红 | 经营、估值、市场、流动性 |
| 债券 | 利息、价格变化 | 信用、利率、流动性 |
| ETF | 组合净值变化 | 市场、跟踪误差、折溢价 |
| 期货 | 合约价格变化 | 杠杆、保证金、到期 |
| 期权 | 非线性价格变化 | 时间价值、波动率、杠杆 |

## 中国股票市场结构

A 股是以人民币计价、主要在境内交易所交易的股票集合。常见交易场所包括上交所、深交所和北交所。

板块会影响上市标准、投资者适当性和部分交易规则。常见板块包括沪深主板、科创板、创业板和北交所市场。

> [!NOTE] A 股规则
> 板块规则会修订。回测系统不应仅根据代码写死规则，而应保存交易所、板块、证券状态和规则生效日期。

## 指数、板块与股票池

指数用一组证券和权重描述市场或风格的总体变化。指数点位不是一只可直接买卖的股票；指数基金或股指期货才是相应工具。

股票池是策略在某个时点允许选择的证券集合，需要排除尚未上市、已经退市、长期停牌或不符合条件的证券。

若用今天仍上市的公司回测十年前，你提前知道谁活到了今天，这叫幸存者偏差。正确方法是使用每个历史时点当时可知的成分，称为 Point-in-Time，中文是点时数据。

## 贯穿案例

- 600001.SH：主板制造教学股；
- 000001.SZ：主板银行教学股；
- 300001.SZ：创业板科技教学股。

它们是人为构造的教学证券。代码后缀帮助识别交易所，但不能仅凭后缀推断某天全部规则。

## 完整代码实践

```python
from pathlib import Path
import pandas as pd

data_dir = Path("E:/Study/quant-lab/data")
instruments = pd.read_csv(
    data_dir / "instruments.csv",
    parse_dates=["list_date"],
)

print(instruments)
print("\n按交易所计数：")
print(instruments.groupby("exchange")["symbol"].count())
print("\n按板块计数：")
print(instruments.groupby("board")["symbol"].count())
```

`parse_dates` 把上市日期解析为日期类型；`groupby` 按类别分组；`count` 统计非空证券代码数量。输出只描述教学证券，不代表真实市场数量。

## 常见误区

1. 债券没有风险：至少存在信用、利率和流动性风险。
2. ETF 就是股票：交易方式类似，底层却是资产组合。
3. 指数涨了就能按指数收益买到：指数本身不可直接成交。
4. 今天的股票池可以回填历史：会产生幸存者偏差。
5. 创业板只是代码不同：还涉及上市与交易制度差异。

## 本章总结

- 不同资产的权利、现金流和风险来源不同。
- 股票总收益包含价格变化和分红。
- A 股由多个交易所和板块组成，规则不能一概而论。
- 指数是统计工具，股票池必须按历史时点构造。
- 教学股票池用于验证流程，不用于荐股。

## 零基础比较练习：同样“涨 10%”为何风险不同

假设股票、债券 ETF 和期货账户都显示某段时间收益 10%，不能因此认为它们相同：

| 问题 | 股票 | 债券 ETF | 期货 |
|---|---|---|---|
| 权利本质 | 企业剩余所有权 | 一篮子债权资产份额 | 标准化合约 |
| 主要风险 | 企业、估值、市场 | 利率、信用、流动性 | 标的、杠杆、保证金 |
| 现金流 | 分红不确定 | 利息和净值变化 | 每日结算盈亏 |
| 是否可能杠杆放大 | 融资时可能 | 通常较低但非零 | 保证金天然带杠杆 |

### A 股代码如何进入项目

```python
def normalize_symbol(code: str, exchange: str) -> str:
    clean_code = code.strip().zfill(6)
    suffix = {
        "SSE": "SH",
        "SZSE": "SZ",
        "BSE": "BJ",
    }[exchange]
    return f"{clean_code}.{suffix}"

assert normalize_symbol("1", "SZSE") == "000001.SZ"
```

证券代码必须按字符串处理，否则前导零会消失。

### 股票池的三个层次

1. 市场全集：历史上存在过的全部证券。
2. 历史可交易集合：在日期 $t$ 已上市、未退市并满足状态要求。
3. 策略候选集合：从历史可交易集合中再按流动性、行业或因子筛选。
